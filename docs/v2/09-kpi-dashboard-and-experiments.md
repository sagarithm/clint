# V2 KPI Dashboard and Experiments Framework

## Purpose
Run growth as an evidence-driven system with clear accountability.

## Funnel KPI Stack

### Top of Funnel
- leads_discovered
- leads_qualified
- qualification_rate
- source_freshness

### Mid Funnel
- messages_generated
- messages_sent
- delivery_success_rate
- open_rate (if available)
- reply_rate
- positive_reply_rate

### Bottom Funnel
- meetings_booked
- qualified_meetings
- proposals_sent
- deals_closed
- revenue_attributed

## Dashboard Views
- Source performance view
- Campaign performance view
- Inbox health view
- Template performance view
- Weekly executive summary

## Experiment Framework

### Hypothesis Format
- If we change X for segment Y, metric Z should improve by N%

### Test Design
- Single primary variable per test
- Minimum sample threshold before decision
- Fixed test window and stop conditions

### Decision Policy
- Promote winner only if uplift is statistically meaningful and stable
- Roll back variants with negative quality impact
- Archive learnings with versioned notes

## Weekly Review Pack
- Top 3 winning sources
- Top 3 losing sources
- Top performing templates and opening lines
- Deliverability risk summary
- Next week action plan with owners

## Threshold Alerts
- sudden drop in positive replies
- inbox bounce spike
- source outage
- queue backlog growth

## Attribution Model
- First-touch source
- Last-touch sequence step
- Assist contribution from follow-up stages
- Revenue attribution per source-template pair

## KPI Targets (Starter)
- Qualification rate: > 20%
- Positive reply rate: > 3%
- Meeting booking rate: > 1%
- Qualified meeting ratio: > 60%

Adjust by niche and campaign maturity over time.
